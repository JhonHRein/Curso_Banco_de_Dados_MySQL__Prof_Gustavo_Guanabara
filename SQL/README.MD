# Scripts SQL

Esta pasta contém os scripts SQL desenvolvidos durante o curso
**Banco de Dados MySQL** do *Curso em Vídeo*, ministrado pelo
professor Gustavo Guanabara.

## Ordem recomendada de execução

1. `01_criacao_do_banco.sql`
2. `02_criacao_de_tabelas.sql`
3. `03_relacionamentos.sql`
4. `04_inserts.sql`
5. `05_consultas.sql`

Cada arquivo representa uma etapa do aprendizado, indo desde
a criação do banco de dados até consultas utilizando JOINs.
