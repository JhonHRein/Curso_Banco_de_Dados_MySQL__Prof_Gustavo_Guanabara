# Scripts SQL

Esta pasta contém os scripts SQL desenvolvidos durante o curso
**Banco de Dados MySQL** do *Curso em Vídeo*, ministrado pelo
professor Gustavo Guanabara.

Obs: As praticas usadas nesse repositorio nao segue os exercicios usados pelo professor guanabara, apenas uso os ensinamentos e inspiraçoes do curso e aplico do meu proprio jeito, usando desafios criados por I.A para meus estudos.

## Ordem recomendada de execução

1. `01_criacao_do_banco.sql`
2. `02_criacao_de_tabelas.sql`
3. `03_relacionamentos.sql`
4. `04_inserts.sql`
5. `05_consultas.sql`

Cada arquivo representa uma etapa do aprendizado, indo desde
a criação do banco de dados até consultas utilizando JOINs.

Obs: Os codigos sao exclusivamente escritos por mim. 
A formataçao dos comentarios sobre as linhdas de codigos, sao feitas por I.A, para melhor leitura e aprendizado dos demais Devs iniciantes.
